How This API Works

This server provides an api for interacting with a mysql database. The database must be configured as dictated in the 'Format of Database.txt' file.

Auth Methods

This api provides two post methods:
  • Login
  • Register
These methods take in two fields:
  • username
  • password
A successful login will result in the user being redirected to the '/game' page. A failure will HTTP status code 400.
A successful registration will return a code 200. A failure will return code 400.

Background Methods

This api deals with interacting with possible game backgrounds.
This api provides two get methods and a post method.
The '/' get method retreives a list of backgrounds available for use.
The '/:id' get method will retreive a background whose name is represented by the ':id' parameter.
The '/upload' post method will allow users to upload an image submitted by a request with a 'file' field that points to the image.
